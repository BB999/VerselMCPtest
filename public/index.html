🤖 Vercel MCP Hello Server

このサーバーはMCP (Model Context Protocol) APIを提供しています。

📡 API エンドポイント

  • /api/server - サーバー情報
  • POST /api/server - MCP API呼び出し

🛠️ 利用可能なツール

  • say_hello - こんにちはと挨拶を返します

📖 使い方例

curl -X POST /api/server \
  -H "Content-Type: application/json" \
  -d '{
    "method": "tools/call",
    "params": {
      "name": "say_hello",
      "arguments": {"name": "太郎"}
    }
  }'
